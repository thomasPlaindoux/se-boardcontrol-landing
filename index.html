Installation

Up and running in 60 seconds

One command installs everything. No Docker, no cloud account, no configuration file to edit manually.

1

Run the installer

One curl command clones the repo, creates a Python virtual environment, and installs all dependencies.

2

Connect Salesforce

Open Settings and click "Connect Salesforce". A browser OAuth flow will log you into your Org62 in 30 seconds.

3

Connect Google Calendar

Optional but recommended. One-click OAuth — grants read access to your calendar events only.

4

Add your Gemini key

Paste a free Gemini API key to unlock AI features. Takes 30 seconds to generate at Google AI Studio.

curl -sL https://raw.githubusercontent.com/thomasPlaindoux/sf-se-boardcontrol/main/dashboard/install.sh | bash

Prerequisites: macOS (Apple Silicon or Intel)  ·  Python 3.10+ (brew install python3)  ·  Git (included with Xcode CLT)  ·  Salesforce CLI (npm install -g @salesforce/cli)

Privacy & Security

Your data stays on your Mac

SE Board Control is a local application. No telemetry, no cloud sync, no third-party data processing beyond what you explicitly configure.

Local-only storage

All notes, configurations, and preferences are stored in your browser's localStorage. Nothing is sent to any external server by the app itself.

100% local

Standard OAuth only

Salesforce and Google connections use the official OAuth 2.0 flows. Credentials are stored in your home directory, never transmitted elsewhere.

OAuth 2.0

Open source

The full source is on GitHub. Read every line, audit every API call, fork it, customize it. No obfuscated binaries, no hidden network requests.

MIT Licensed
